# Release Notes

## v0.1.0-beta

初回ベータ版です。

### 追加された機能

* macOSからKIT学内オンデマンドプリンターへPDFファイルを送信できるようにしました。
* PostScriptファイルを直接送信できるようにしました。
* PDFファイルをPostScriptへ自動変換してから送信する機能を追加しました。
* `iogate3.kanazawa-it.ac.jp` へHTTP Basic認証付きのIPP 1.0 Print-Jobとして送信する処理を実装しました。
* 初期設定として、KIT ADユーザー名を `~/.config/kitprint/config.json` に保存できるようにしました。
* `kitprint --setup` によるユーザー設定機能を追加しました。
* `kitprint /path/to/file.pdf` の形式で、任意のパスにあるPDFを印刷できるようにしました。
* `-n` オプションでジョブ名を指定できるようにしました。
* `-u` オプションで一時的に別のユーザー名を指定できるようにしました。
* `install.sh` により、`kitprint` コマンドを `~/bin` に登録できるようにしました。

### 現在の制限

* 現時点では、基本的にカラー・片面印刷として送信されます。
* 白黒印刷、両面印刷、部数指定には未対応です。
* Word、PowerPointなどのファイルを直接送信することはできません。PDFに書き出してから使用する必要があります。
* パスワードは保存されません。印刷時に毎回入力する必要があります。
* 学内ネットワークまたはVPN接続が必要です。
* PDFによっては、レイアウトや用紙サイズの影響で印刷結果が崩れる可能性があります。

### 今後の予定

* 白黒印刷への対応
* 両面印刷への対応
* 部数指定への対応
* より分かりやすいエラー表示
* GUI版の作成
* macOSアプリケーション化
* 可能であればWordなどの印刷画面から直接送信できる仕組みの検討

### 注意

このバージョンはベータ版です。今後、コマンドの使い方や設定ファイルの形式が変更される可能性があります。
